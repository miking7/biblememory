About Bible Memory

Helping believers hide God's Word in their hearts

Hi, I'm Michael Engelbrecht, and I built Bible Memory to help believers hide God's Word in their hearts.

Why I Built This

I wanted a Scripture memorization tool that:

  • Works offline (no internet required)
  • Syncs across my devices
  • Uses proven spaced repetition (not random review)
  • Looks beautiful and feels modern
  • Respects my privacy (no tracking, no ads)

Most Bible apps focus on reading, not memorization. The memorization tools I found were either outdated, web-only (no offline), or locked behind paywalls.

So I built the tool I wanted to use.

Open Source Commitment

Bible Memory is 100% open source. That means:

  • You can see exactly what the code does
  • No hidden tracking or data selling
  • Free forever (no premium tiers)
  • Community can contribute improvements

The technology stack: Vue 3, TypeScript, IndexedDB, Progressive Web App, and spaced repetition algorithms.
